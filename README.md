# Spurningaleikur

Spurningaleikur is a client-only trivia quiz platform supporting multiple categories and quiz types. 

## Features
- Category selection (e.g., Brawl Stars, Dogs)
- Two quiz types: Type the Name and Multiple Choice
- Localized UI (Icelandic)
- Easily extensible with new categories and data

## Usage
1. Select a category and quiz type on the main page.
2. Play the quiz and get instant feedback.
3. Add new categories by editing `categories.json` and adding a data file (e.g., `dogs.json`).

## Development
- All static files, no backend required.
- Place images in the appropriate `images/<category>/` folder.
